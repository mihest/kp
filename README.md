# Курсовой проект Next-js (React) + laravel (Rest api)



## Запуска web-интерфейса

### 1. Требования

- Node.js 21.7.1

### 2. Запуск
- cd web
- npm install
- npm run dev



## Запуск api

### 1. Требования

- PHP 8.3
- composer 2.7.2

### 2. Запуск

- cd php
- composer install
- перед запуском настроить .env
- php artisan serve (Запуск api на локальном сервере)
